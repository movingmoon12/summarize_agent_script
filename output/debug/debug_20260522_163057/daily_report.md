### 2026年度概念验证入库项目（第二轮）——核心信息

- **目标**：面向全校科研人员，遴选2项概念验证项目，每项资助 **50万元**。
- **资格**：
  - 负责人：苏州大学在职科研人员，博士或中级以上职称，无知识产权纠纷。
  - 项目：有扎实基础、未转化、未成立公司、无社会资本，可在江都高新区（含苏州创新飞地）落地。
- **流程**：
  - 填报《入库征集表》+ 附件材料 → 学院审核 → 专家评审公示 → 立项。
- **周期**：原则上 ≤ 1年，须在 **2027年2月底前** 在江都高新区（含苏州创新飞地）注册公司。
- **经费拨付**：立项后40%；中期检查且首批用完后拨付60%。
- **场地**：落地公司可优先免费使用办公场地。
- **知识产权**：
  - 验证阶段知识产权与江都高新区共享。
  - 若成功出库并成立公司 → 江都高新区无偿归还共享知识产权。
  - 若退出或未通过 → 知识产权出售/拍卖优先偿还江都高新区投入。
- **材料提交**：
  - **电子版**：5月28日17:00前，发至 `yurenyan@suda.edu.cn`，邮件命名 `概念验证+姓名+项目名称`。
  - **纸质版**：一式六份，5月29日17:00前交至独墅湖校区704楼303办公室。
- **联系人**：俞仁琰，13862562051，65882903。

### 核心信息

- **申报项目**：可持续发展国际合作科学计划（SDIC）2026年度第一批补充指南，含2类组织间合作项目。
